
# MVP DEFINITION

## MVP Objective
Deliver a functional data index proving value for **one country (Benin)** and **one vertical (Agriculture)**.

## INCLUDED IN MVP
- Agriculture Data Index (ADI)
- Commune-level aggregation
- 3 strategic crops
- REST API
- Admin ingestion pipeline
- Simple analytics dashboard

## EXCLUDED FROM MVP
- Payments
- ML predictions
- User-generated content
- Multi-country support

## MVP USERS
- NGOs
- Research labs
- Pilot institutional partners

## VALIDATION CRITERIA
- Data consistency
- API usability
- First external user adoption
